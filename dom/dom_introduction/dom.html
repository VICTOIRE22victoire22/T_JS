Ce document présente les notions de base pour manipuler le DOM avec JavaScript.
Chaque section est dédiée à une méthode ou un concept spécifique.

Sélectionner des éléments du DOM

Sélectionner des éléments du DOM

Les sélecteurs permettent de cibler des éléments HTML dans le DOM pour les manipuler.

  • getElementById : Sélectionne un élément par son ID
  • getElementsByClassName : Sélectionne des éléments par leur classe (HTMLCollection)
  • getElementsByTagName : Sélectionne des éléments par leur nom de balise (HTMLCollection)
  • querySelector : Sélectionne le premier élément correspondant à un sélecteur CSS
  • querySelectorAll : Sélectionne tous les éléments correspondant à un sélecteur CSS (Nodelist)
Syntaxe : getElementById('nom_id') getElementByClasssName('nom_de_la_classe') getElementByTagName('nom_de_la_balise') querySelector('selecteur_css') querySelectorAll('selecteur_css')

Parcourir les éléments du DOM

Parcourir les éléments du DOM

Ces méthodes permettent de naviguer entre les éléments du DOM, comme les enfants, les parents ou les frères et sœurs.

  • parentElement : Sélectionne le noeud Elément parent d'un élément
  • children : Sélectionne les noeuds Eléments enfants d'un élément
  • firstElementChild : Sélectionne le premier noeud Elément enfant d'un élément
  • lastElementChild : Sélectionne le dernier noeud Elément enfant d'un élément
  • nextElementSibling : Sélectionne le noeud Elément frère suivant d'un élément
  • previousElementSibling : Sélectionne le noeud Elément frère précédent d'un élément
Syntaxe : element.parentElement element.children element.firstElementChild element.lastElementChild element.nextElementSibling

Modifier le contenu du DOM

Modifier le contenu du DOM

Ces méthodes permettent de modifier le contenu ou la structure des éléments HTML.

  • textContent : Modifie le contenu textuel d'un élément
  • innerHTML : Modifie le contenu HTML d'un élément
  • innerText : Modifie le texte visible d'un élément
Syntaxe : element.textContent element.innerHTML element.innerText

Modifier les classes du DOM

Modifier les classes du DOM

Ces méthodes permettent de manipuler les classes CSS des éléments HTML.

  • classList.add : Ajoute une classe à un élément
  • classList.remove : Supprime une classe d'un élément
  • classList.toggle : Ajoute ou supprime une classe d'un élément selon sa présence
  • classList.contains : Vérifie si un élément a une classe spécifique
  • classList.replace : Remplace une classe par une autre sur un élément
Syntaxe : element.classList.add('nomClasse') element.classList.remove('nomClasse') element.classList.toggle('nomClasse') element.classList.contains('nomClasse') element.classList.replace('ancienneClasse', 'nouvelleClasse')

Modifier le style du DOM

Modifier le style du DOM

Ces méthodes permettent de changer le style d'un élément.

  • style : Modifie le style CSS d'un élément
  • setAttribute : Modifie un attribut d'un élément
  • getAttribute : Récupère la valeur d'un attribut d'un élément
  • removeAttribute : Supprime un attribut d'un élément
Syntaxe : element.style.property = 'valeur' element.setAttribute('attribut', 'valeur') element.getAttribute('attribut') element.removeAttribute('attribut')

Créer des éléments dans le DOM

Créer des éléments dans le DOM

Ces méthodes permettent de créer de nouveaux éléments dans le DOM.

  • createElement : Crée un nouvel élément HTML
  • createTextNode : Crée un nouveau noeud de texte
  • appendChild : Ajoute un enfant à un élément
  • insertBefore : Insère un élément avant un autre élément
Syntaxe : document.createElement('nomBalise') document.createTextNode('texte') element.appendChild(nouvelEnfant) element.insertBefore(nouvelEnfant, elementExistant)
